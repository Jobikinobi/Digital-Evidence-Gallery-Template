### 1.4 EXIF Metadata Explained

**What Gets Extracted from Photos**:
```csv
FileName,DateTimeOriginal,Make,Model,ISO,FNumber,ShutterSpeed,FocalLength,ImageWidth,ImageHeight,GPSLatitude,GPSLongitude
IMG_001.JPG,2023:08:15 14:32:00,Canon,EOS R5,400,2.8,1/500,85mm,6720,4480,31.7619,-106.4850
```

**Key Fields**:
- `DateTimeOriginal` - When photo was taken (critical for timeline!)
- `Make/Model` - Camera identification
- `ISO/FNumber/ShutterSpeed` - Camera settings
- `GPSLatitude/GPSLongitude` - Location (if available)

**Missing EXIF** (Common Scenarios):
- Screenshots: No EXIF data (will show "Unknown")
- Edited photos: EXIF may be stripped
- Downloaded images: Often missing EXIF

### 1.5 PDF Metadata Explained

**What Gets Extracted from Documents**:
```csv
FileName,CreationDate,ModDate,Author,Creator,Producer,PageCount
report.pdf,2023-08-15,2023-08-20,Detective Smith,Adobe Acrobat,Adobe PDF,15
```

**Key Fields**:
- `CreationDate` - When PDF was created (timeline)
- `Author` - Who created document
- `PageCount` - Total pages in PDF

### 2.3 Optimization Settings Explained

**Image Optimization**:
```python
resize_percent = 0.5      # 50% of original resolution
jpeg_quality = 75         # High quality (visually lossless)
thumbnail_size = (150,150)
thumbnail_quality = 40    # Lower (just for preview)
```

**Why 50% Resolution?**:
- Original: 6720Ã—4480px (30 megapixels)
- Optimized: 3360Ã—2240px (7.5 megapixels)
- Still high-res enough for screens (1920Ã—1080 = 2 megapixels)
- 95% file size reduction
- Retains all image quality for viewing

**Document Optimization**:
```python
dpi = 150                 # Screen-optimized (not print)
jpeg_quality = 60         # Aggressive compression
max_pages_huge = 50       # PDFs > 20 MB: first 50 pages
max_pages_large = 75      # PDFs 5-20 MB: first 75 pages
max_pages_small = 100     # PDFs < 5 MB: up to 100 pages
```

**Why 150 DPI?**:
- 72 DPI: Blurry, unreadable
- 150 DPI: â­ Perfect for screens (our choice)
- 200 DPI: Higher quality, larger files
- 300 DPI: Print quality, huge files
